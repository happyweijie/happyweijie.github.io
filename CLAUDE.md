# happyweijie.github.io

Wei Jie's personal portfolio — a static site (plain HTML/CSS/JS) served on GitHub Pages.

## Design Context

Before any UI or design work, read:

- **PRODUCT.md** — strategy: brand register, audience (recruiters), positioning ("a builder who ships"), belief ladder (internships > projects > activities > films), anti-references, design principles.
- **DESIGN.md** — the visual system: "The Live Terminal Session" north star, palette (Prompt Green / Signal Grey over near-black), flat mono type scale, named rules (Two Voices, One Window, Plausible Shell, Flat Scale).

Key constraints: palette is defined once in `index.css` custom properties; the page must stay readable with no JS; every animation needs a reduced-motion fallback; mobile responsiveness is a first-class requirement.
